100,000 boxes, culled where they live

A @compute pass tests every instance's bounding sphere against the frustum, compacts the survivors into the instance stream with one atomicAdd, and writes the draw call's own argument buffer — gpu-draw-indexed-indirect! then draws exactly the visible count. The CPU contributes 176 bytes of uniforms and a 20-byte argument reset per frame; it never inspects an instance. Needs a WebGPU browser (current Chrome / Edge / Safari). See gpu-cull.ss.
