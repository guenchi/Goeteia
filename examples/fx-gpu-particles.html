100,000 particles, zero CPU physics

Transform feedback: the state lives in two GPU buffers and a vertex shader (fx-tf-program!) integrates one into the other each frame — the shader IS the physics, rasterizer discard keeps the update pass invisible — then a point-sprite pass draws the result and the buffers swap. Scheme's whole frame is a handful of command words. Needs WebGL 2. See fx-gpu-particles.ss.
