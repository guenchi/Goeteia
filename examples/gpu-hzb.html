What the walls hide never draws

Three walls render first; the depth buffer reduces into a max-mip pyramid (gpu-hzb!, compute), and the cull kernel tests all 30,000 boxes against the frustum AND the pyramid — a box whose nearest point is farther than everything already drawn over its screen footprint never reaches the vertex shader. Add #nocull to the URL and reload: the image must be pixel-identical, only slower — occlusion culling that changes the picture is a bug. Needs a WebGPU browser. See gpu-hzb.ss.
